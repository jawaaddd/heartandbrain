# heartandbrain

A self-improvement android app that will be evolving over time. The first feature that's been added is taking a self-recorded vlog, transcribing it, and using an LLM to find key ideas that I may want to remember in the future and displaying them for me to look back on.

## Features:
- [x] Vlog parser and motivational quote finder (a reminder of the things I say to myself)
- [ ] Goals/commitments list, and dynamic progress trackers
- [ ] Automatic time blocking and personal schedule builder

## Development
This is the first app that I'm heavily using AI for (Claude Code). The focus is on rapid development to get the app to a stage where I'll be using it often for myself, and for experimenting with AI development workflows and AI agents.
For a showcase of my programming foundations, software design, and hard skills, see my chocolate-hosting project.
